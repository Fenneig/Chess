ТЗ
Игра в шахматы
1. Возможность играть на одном компьютере или используя сеть интернет для двух.
2. Два режима игры: для новичков(с подсветкой возможных ходов у фигур) и обычный(без подсветки).
3. Менять расположение фигур снизу для текущего игрока(если игра ведется на одном ПК) или статично снизу(если используется интернет). 
4. Реализация на Windows Forms.
5. Реализация на Unity3D.
