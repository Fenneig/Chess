Данный файл является чем-то средним между ТЗ, документом обновлений и моими мыслями по проекту Шахматы.

ТЗ
Игра в шахматы
1. Возможность играть на одном компьютере или используя сеть интернет для двух.
2. Менять расположение фигур снизу для текущего игрока(если игра ведется на одном ПК) или статично снизу(если игра ведется на разных пк). 
3. Реализация на Windows Forms.
4. Реализация на Unity3D.
